# useful-scripts
some stuff worth keeping in path


- gitwhat - revives a specified (or current) commit changeset as changes before they were committed (so it shows up as changes in your editor)
- gitrecent - lists recently modified local branches and offers to merge them to current one
- speechbubble - runs whisper in a docker container to record what's being said and type it into current cursor on linux
